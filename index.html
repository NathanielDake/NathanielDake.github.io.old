GTOFFERZ

KeleAccess Gem

KeleAccess Gem is a Ruby Gem that I created as an API client in order to access the Bloc API. Bloc's API provides an external facing JSON Web Token authorized gateway to the Bloc application.KeleAccess was created in order to manage the low level details of making requests and handling responses.

WikiJunkie

I created WikiJunkie as an application that allows users to create public and private Markdown-based wikis. Several of the main features are: The creation of user roles, the ability to create public and private wikis, the implementation of markdown syntax, and the user authentication.

It is built with Ruby on Rails application and is based on CRUD principle. This application incorporates several Ruby Gems such as Devise for user authentication and account confirmation, Pundit for role authorization (standard, premium, or admin), Stripe for user account upgrade from standard (free) to premium (paid), Redcarpet to parse text into markdown syntax,RSpec for test driven development, and Faker for seeding data.

YourForum

YourForum was created as a Ruby on Rails application that is based on CRUD principles and designed to replicate functionality of the website Reddit.

It allows users to create topics, posts, and comments, as well as favorite posts, and up/down vote on posts. Additionally I created an different user roles (admin and standard), while always utilizing proper object oriented design principles and test driven development.

Robotic Prosthetic Glove
